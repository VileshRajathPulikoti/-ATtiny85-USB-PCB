# ATtiny85 USB PCB

Designed by Venkronix – Volt Vision

## Overview
This project is a compact USB-based PCB built around the ATtiny85 microcontroller. The board is designed for embedded development and USB communication.

## Features
- ATtiny85 Microcontroller
- USB Interface
- Zener diode protection for USB data lines
- LED indicator
- External pin headers for IO expansion
- Compact PCB layout

## Hardware Components
- ATtiny85
- USB Interface pads
- 3.6V Zener Diodes
- LED Indicator
- Resistors
- Pin Headers

## Tools Used
- KiCad 9
- PCB 3D Viewer
- Embedded C Programming
